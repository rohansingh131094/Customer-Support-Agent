Key Engineering Decisions

Every choice made deliberately — and why.

🔩
Raw SDK, No Framework
No LangChain, no abstraction layer. Direct Anthropic SDK calls give full control over the tool loop, streaming, and error handling — and avoid framework lock-in.
⚡
Two-Model Architecture
Haiku classifies intent on every message — fast and cheap. Sonnet executes — powerful and accurate. Using a smaller model for routing keeps latency and cost low.
📦
Self-Contained Intents
Each intent owns its system prompt and tool subset. The agent never sees tools it doesn't need, reducing hallucination risk and keeping prompts focused.
🚿
Filler Text Suppression
Text generated before a tool call ("Let me look that up!") is buffered and discarded. Only the final, informed response is streamed — no noise, no filler.
